# 💰 Student Budget Planner

A web-based budgeting app built for university students to track expenses, manage monthly budgets, and visualise spending patterns.

[![Live Demo](https://img.shields.io/badge/LIVE%20DEMO-Nuwebspace-6366f1?style=for-the-badge)](https://w25005997.nuwebspace.co.uk/kv6027/StudentBudgetPlanner/)

---

## Features

- Login and signup with account validation
- Add, edit and delete expenses across 8 categories
- Monthly budget with traffic light warning system
- Spending charts — pie chart and daily trend
- Category budget sliders
- Profile page with avatar and sticker picker
- Spending insights and 6-month history
- Dark and light theme
- Multi-currency support (GBP, USD, EUR)
- Export and import data as JSON backup

---

## Tech Stack

React 18 · Vite · Recharts · Framer Motion · Lucide React · localStorage

---

## Run Locally

```bash
git clone https://github.com/Amarachiiroha/Student-Budget-Planner.git
cd Student-Budget-Planner
npm install
npm run dev
```
